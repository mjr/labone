## Como compilar e rodar cada programa?

1. Entre na pasta do programa (ex: question_one).
2. Compile os arquivos.
3. Execute o programa.

```console
cd question_one/
make
./bin/main --options
```

1. Segunda questão.

```console
g++ -o main -O0 main.cpp
```
